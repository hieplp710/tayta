== Description ==
This plugin allows users to easily add a simple user registration form and login form  anywhere on their site using simple shortcode.
And provide setting to 'Auto Login After Registration'. 
 
Follow the follwing steps -

* Upload the folder "auto-login-after-registration" to "/wp-content/plugins/"
* Activate the plugin through the "Plugins" menu in WordPress
* Enable/ disable 'Auto Login After Registration' setting by going WP-admin ->Auto Login after Register 
* For registrtion form use below shortcode  
`
[registration-form]
`
For login form use below  shortcode  
`
[login-form]
`
